# Playwright JS/TS Automation Testing Learning Project

This repository contains my hands-on learning and practice work while studying Playwright automation testing using JavaScript and TypeScript.

The learning material is mainly based on the course:

**Playwright JS/TS Automation Testing from Scratch & Framework** by Rahul Shetty.

## Topics Covered

- Playwright fundamentals
- JavaScript & TypeScript basics for automation
- UI Automation Testing
- API Testing with Playwright
- Network request & response interception
- Playwright Inspector & Trace Viewer
- Codegen / Record & Playback
- Page Object Model (POM)
- Playwright with Cucumber BDD
- Mocha Framework integration
- Visual Testing
- Mobile device emulation
- CI/CD execution
- Docker-based test execution
- AI Agents, MCP Servers & GitHub Copilot usage with Playwright

## Tech Stack

- Playwright
- JavaScript
- TypeScript
- Node.js
- Git & GitHub

## Purpose

The goal of this repository is to practice real-world automation testing concepts, improve framework design skills, and build stronger QA automation engineering experience.
